# Vigezo katika programu ya Python
Wakati mwingine unapoandika jina ya vigezo kwa lugha ya kiingereza, majina haya hayana mantiki. Tumia kiswahili katika majina ya vigezo. Matumizi ya majina ya kiswahili itafanya programu yako ionekane viruzi. Kwa mfano:

``` Python
# Variable name in English
total_number_of_people_invited = 2450

# Variable name in Swahili
jumla_ya_waalikwa = 2450
```
